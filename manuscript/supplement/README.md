# Supplement: first-sampling chronology (secondary only)

This supplement reports earliest sample-collection-date order by case group.
It is **not** a transmission-direction analysis.

## Confirmed wards (n=18)

See `supp_first_sampling_summary_confirmed.csv` and `supp_first_sampling_order.csv`.

Interpretation constraints (Gu et al. 2026):
- Confirmation requires ≥1 likely nosocomial inpatient; non-inpatient-only units are non-confirmed by definition.
- Dates are sample collection dates, not infection or clinical detection dates.
- Non-inpatient combines staff and outpatients in the source plot table.
- Regional Fisher tests are omitted from the main manuscript as severely underpowered and non-central.
